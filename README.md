# AUV Procurement Tracker

Live status and costings for the AUV fit-out procurement — open it, sign in, and edit
directly. Changes save instantly and everyone sees the same up-to-date data.

**Link:** https://witty-mushroom-0d743f603.7.azurestaticapps.net/

## Signing in

1. Open the link above.
2. Click **Sign in with Microsoft** (top right).
3. Sign in with your Skyports work account (`@skyports.net`).
4. Give it a second — the table should load and the top right will say
   **"Synced with SharePoint at [time]"**.

If you instead see a red error banner, or Microsoft says you're not authorized to
sign in, send a screenshot to Ryan — you likely need to be added to the group that's
allowed to use this app.

## What you'll see

- One row per procurement item (sensors, network kit, FF&E, GSE, etc.)
- **Status** — Not Started / Pending / On Hold / Done, shown as a coloured pill
- **Update** — a free-text note on where things currently stand
- **Next action** — what needs to happen next, and who's doing it
- **Est. cost** / **Confirmed cost** — in AED
- **Cost status** — Estimate / Quote Received / PO Issued / Invoiced / Paid
- Four tiles across the top: total estimated cost, total confirmed cost, how many
  items are tracked, and how many are Done

Use the **status filter** and **search box** above the table to narrow it down.

## Making changes

Click into any field and edit it — status and cost status are dropdowns, everything
else is free text or a number. It saves automatically the moment you click away; there's
no save button and nothing to submit. Your edit is visible to everyone else immediately.

There's currently no way to rename an item itself (e.g. fix a typo in the item name)
from within the app — if that's needed, ask Ryan to edit it directly in the underlying
SharePoint list.

## Something wrong?

Screenshot the error banner (it shows the real error message, not a generic failure)
and send it to Ryan.
